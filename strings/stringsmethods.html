toUpperCase(): Returns a new string converted to uppercase.
const text = "Hello, World";
alert(text.toUpperCase());
toLowerCase(): Returns a new string converted to lowercase.
alert(text.toLowerCase());
trim(): Removes leading and trailing whitespace characters from the string.
document.getElementsByClassName("trim")[0].addEventListener("click", function () {
alert(text.trim());
})
            
slice(start, end): Extracts a section of the string and returns a new string.
document.getElementsByClassName("slice")[0].addEventListener("click", function(){
  alert(text.slice(0, 5)); // Output : Hello
})
            
substring(start, end):(Less common than slice) Similar to slice, extracts a section of the string.
document.getElementsByClassName("sub_string")[0].addEventListener("click", function(){
  alert(text.substring(0, 5)); // Output : Hello
})
            
concat(string1, string2, ...):Joins multiple strings together and returns a new string.

indexOf(substring, fromIndex):

  • This method searches for the first occurrence of a specified substring within a string.
  • If the substring is found, it returns the index (position) where it starts.
  • If the substring is not found, it returns -1.
  • The fromIndex parameter (optional) specifies where to start the search from within the string (defaults to the beginning).

lastIndexOf(substring, fromIndex):

  • This method searches for the last occurrence of a specified substring within a string.
  • If the substring is found, it returns the index (position) where the last occurrence starts.
  • If the substring is not found, it returns -1.
  • The fromIndex parameter (optional) specifies where to start the search from within the string (defaults to the end, searching backward).
Code example in the last JS file.

includes(substring, fromIndex):

  • This method checks if the specified substring exists anywhere within a string.
  • It returns true if the substring is found, and false otherwise.
  • The fromIndex parameter (optional) specifies where to start the search from within the string (defaults to the beginning).
