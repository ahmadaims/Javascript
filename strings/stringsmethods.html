Accessing Characters

charAt(index): Returns the character at a specific index within the string.
charCodeAt(index): Returns the Unicode character code at a specific index.
Click on the below code box to work this code. 

document.getElementsByClassName('charAt')[0].addEventListener("click",function(){
const message = "Hello, world!";
alert(message.charAt(0)+"\n"+message.charCodeAt(7)); // Output : H 119
})

toUpperCase(): Returns a new string converted to uppercase.
const text = "Hello, World";
alert(text.toUpperCase());
toLowerCase(): Returns a new string converted to lowercase.
alert(text.toLowerCase());
trim(): Removes leading and trailing whitespace characters from the string.
document.getElementsByClassName("trim")[0].addEventListener("click", function () {
alert(text.trim());
})
            
slice(start, end): Extracts a section of the string and returns a new string.
