Javascript Variables

Variables are Containers for Storing Data

JavaScript Variables can be declared in 4 ways:

  • Automatically
  • Using var
  • Using let
  • Using const

Note:

The var keyword was used in all JavaScript code from 1995 to 2015.
The let and const keywords were added to JavaScript in 2015.
The var keyword should only be used in code written for older browsers.

Example using var

var x = 5;
var y = 6;
var z = x + y;
z = 11

Example using let

let x = 5;
let y = 6;
let z = x + y;
z = 11

Example using const

const x = 5;
const y = 6;
const z = x + y;
z = 11

Mixed Example

const price1 = 5;
const price2 = 6;
let total = price1 + price2;
total = 11

The two variables price1 and price2 are declared with the const keyword.
These are constant values and cannot be changed.
The variable total is declared with the let keyword.
The value total can be changed.

When to Use var, let, or const?

  1. Always declare variables
  2. Always use const if the value should not be changed
  3. Always use const if the type should not be changed (Arrays and Objects)
  4. Only use let if you can't use const
  5. Only use let if you can't use const
  6. Only use var if you MUST support old browsers.

JavaScript Identifiers

In JavaScript, every variable must have a distinct name, known as an identifier. These identifiers can be concise, like x and y, or more descriptive, such as age, sum, or totalVolume.
Here are the fundamental rules for creating variable names:

  • Identifiers may consist of letters, digits, underscores, and dollar signs.
  • They must begin with a letter.
  • While they can begin with $ and _, we won't use these in this guide for simplicity.
  • JavaScript is case-sensitive, meaning y and Y represent different variables.
  • Reserved words, such as JavaScript keywords, cannot be used as identifiers.
  • By following these guidelines, you'll create clear and understandable variable names in your JavaScript code.
